## ようこそ！

これはHugoで作成したブログの最初の投稿です。

### Hugoとは？

HugoはGo言語で書かれた静的サイトジェネレーターです。高速なビルド速度と様々なテーマをサポートしています。

### 次のステップ

- もっと投稿を書く
- テーマをカスタマイズする
- デプロイする

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Hugo!")
}
```
